# README

## 환경
- node v20.10.0
- Vite v5.2.0
- React v18.2.0 
- Typescript v5.2.2


## 앱 실행 순서
1. node 버전 확인
```
node -v
```
2. vite 설치
```
npm install -g create-vite
```
3. 현재 폴더에 React & Typescript 로 구성된 앱 설치
```
create-vite . --template react-ts
```
4. 패키지 설치 & 실행
```
npm install
npm run dev
```
